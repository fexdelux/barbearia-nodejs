# Projeto Babearia

## Funcionalidade
- Cadastro do cliente
- Cadastro de produtos
- cadastro dos serviços
- Agendamento do cliente
- caixa


### Cadastro de cliente

Campos de informações:
- Nome do cliente
- Telefone 
- email
- data de cadastro

### Menu de navegação

Itens: 
- cadastro de cliente
- Cadastro de serviço
- agendamento
- execução do serviço

### Cadastro dos serviços

Campos de Informações:
- Titulo
- Tipo
- Valor

### Agendamento
Campos de Informações:
- Cliente
- Data 
- Hora

### caixa
Campos de Informações:
- Cliente
- serviços (Lista)
- Produtos (Lista)
- Total a pargar
- tipo de pagamento (PIX, cartão ou Dinheiro)
- data da execução
- Feedback (1 a 5)


## Desenvolvimento

Cada funcionalidade será um modulo.

Cada modulo será exportado como Classe.

As classe de cadastro deverá ter os metodos de : 
- Salva
- Alterar
- Listar
- Deletar
